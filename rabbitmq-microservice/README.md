# RabbitMQ in microservice

Run rabbit mq with docker.

```sh
$ docker-compose up -d;
```

Check logs from container
```sh
$ docker-compose logs -f;
```

To access rabbit mq.
[Rabbit MQ UI](!http://localhost:15672/#/)
